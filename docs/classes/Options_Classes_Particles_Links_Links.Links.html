color

color: OptionsColor

Implementation of ILinks.color

  • Defined in src/Options/Classes/Particles/Links/Links.ts:14

Link line color, can be a string or an IColor object. If the value is "random", if blink is true sets a random color each frame, if consent is true a random color is used for all links, if blink and consent are false the color will be a gradient with the two linked particles color
